# AI会議：Youkan カラーシステム定義 - 眼精疲労・頭痛軽減特化型 (v4.1)

---

## 1. 会議の定義

### テーマ
新システム名「**Youkan (羊羹)**」を冠した、眼精疲労と緊張型頭痛に苦しむ晴樹氏のための最終カラーシステム定義。

### ゴール
1.  **「羊羹」を彷彿とさせる深みのある配色**: 脳を落ち着かせる、重厚かつ透明感のあるダークモードの確立。
2.  **Youkan Ergonomic Palette の策定**: 全てのトークンを `youkan-*` 形式で再定義。
3.  **視覚的ストレスの除去**: ハレーション、にじみ、強い光の徹底排除。

### 前提条件
- **名前の由来**: 「管理を味わう」「日を切り分ける」＝ 羊羹の持つ静謐さと密度。
- **憲法優先**: 判断疲れを誘発する刺激的な色（警告色など）を排除し、情報の「気配」として色を扱う。

## 2. キャスティング

1.  **ファシリテーター**: Antigravity
2.  **UX・視覚疲労専門家**: 眼精疲労と頭痛のメカニズムに基づき、配色を検証。
3.  **色彩工学担当**: 羊羹のような「深みのある黒・紫・茶」をデジタル色空間（Tailwind/CSS）で再現。
4.  **JBWOS 思想継承者**: システム名が変わっても「JBWOS」の哲学（判断OS）を損なわないよう監視。

## 3. ラウンド構成（統合要約）

### 【ラウンド1】初期提案：「羊羹」の質感
**色彩工学担当**:
「『羊羹』という名前は、ダークモードにとって完璧な比喩です。高級な羊羹のような、わずかに赤みや紫みを含んだ『深淵な黒』をベースにします（#0a0809）。これにより、単なる無機質なグレーよりも脳がリラックスできます。ハイライトには『琥珀（Amber）』を使い、情報の重要度を熱量として表現します。」

### 【ラウンド2】交差レビュー：頭痛対策の徹底
**UX専門家**:
「晴樹さんの頭痛対策として、フォントのエッジと背景のコントラスト比を 7:1 程度に抑えます。Dark Mode での文字色は純白ではなく `--youkan-text-soft` (#c5c9d1) を標準とし、目の焦点を合わせやすくします。また、背景の『羊羹色』は OLED 画面でのにじみを防ぐため、真黒（#000）を絶対に避け、ピクセルがオフにならない程度の明度を維持します。」

### 【ラウンド3】収束：Youkan セマンティック・トークンの決定
**思想継承者**:
「全ての接頭辞を `youkan-` に統一します。単なる『色』ではなく『役割』で命名することで、実装時の迷いを消します。例えば `youkan-focus` は『今、切り分けているその一切れ』を指し、`youkan-ambience` は『周りの気配』を指します。」

## 4. 最終結論

### 4.1 Youkan Ergonomic Palette (v4.1)

| 変数名 | 役割（意味） | Light Mode | Dark Mode (Night) | 特徴 |
| :--- | :--- | :--- | :--- | :--- |
| `--youkan-bg-base` | 背景（大地） | `#f9fafb` | `#080709` (深紫黒) | 脳を休ませる基盤色 |
| `--youkan-surface` | 面（羊羹の断面） | `#ffffff` | `#16141a` (深灰紫) | 重なりによる整理 |
| `--youkan-text` | 主文（墨） | `#1a1a1c` | `#dcdde1` (乳白) | ハレーション・にじみ防止 |
| `--youkan-muted` | 補足（気配） | `#9ca3af` | `#5a5860` | 周辺視野への刺激低減 |
| `--youkan-primary` | 意志（藍） | `#4f46e5` | `#818cf8` | 視線誘導を優しく行う |
| `--youkan-volume` | 量感（琥珀） | `#f59e0b` | `#b45309/80` | 暗所での穏やかな熱量 |

### 4.2 デザイン・ガイドライン
- **命名規則**: Tailwind Config では `colors.youkan.bg.base` のように構造化。
- **アクセシビリティ**: 全てのテキストは WCAG 2.1 AA 以上のコントラストを維持しつつ、最大コントラスト（刺激）は避ける。
- **余白の呼吸**: 羊羹を美しく並べるように、十分なホワイトスペース（一貫した 16px/24px 単位）を確保する。
