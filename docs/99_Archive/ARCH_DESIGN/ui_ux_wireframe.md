# UI/UX ワイヤーフレーム提案：製造業統合とコンテキスト管理

本ドキュメントでは、製造業プラグインと個人/会社のコンテキスト管理を直感的かつ「判断不要」に実現するためのUI案を提示します。

## 1. コンテキスト対応プロジェクト作成モーダル
会社タブを選択している際に表示される「業務特化型」の作成画面です。

![新規プロジェクト作成モーダル（会社モード）](/C:\Users\doorf\.gemini\antigravity\brain\4b9a65a8-67b4-4b6b-951c-462182743bdd\mfg_project_creation_modal_dark_1769642173523.png)

- **デザイン意図**: 
    - 通常のプロジェクト名に加え、「元請け」「現場名」「摘要」を標準フィールドとして配置。
    - 図面や写真を即座に登録できるドラッグ＆ドロップエリアを設け、現場の情報を一箇所に集約。
    - 入力項目を絞ることで、登録時の「迷い」を排除。

## 2. パーソナル/会社タブ切り替えUI
プロジェクト一覧における「基本」と「会社」の厳格な分離を視覚的に表現します。

![プロジェクト一覧のタブUI](/C:\Users\doorf\.gemini\antigravity\brain\4b9a65a8-67b4-4b6b-951c-462182743bdd\project_list_tabs_ui_dark_1769642192824.png)

- **デザイン意図**:
    - 上部に大きなタブスイッチ（基本/会社）を配置。現在どちらのコンテキストにいるかを色とグロー（発行）効果で強調。
    - 会社タブでは、プロジェクトを会社名（クライアント）ごとに自動セクション分けし、大量の案件を俯瞰しやすく。
    - 右下の「＋」ボタンは、選択中のタブに応じた適切なモーダルを起動。

## 3. 製造業統合ダッシュボード（ソート＆フィルタ）
1日のタスクを「判断不要」に整理し、製造業の負荷を可視化する画面です。

![統合ダッシュボードと表示設定](/C:\Users\doorf\.gemini\antigravity\brain\4b9a65a8-67b4-4b6b-951c-462182743bdd\mfg_dashboard_sorting_ui_dark_1769642221963.png)

- **デザイン意図**:
    - **表示設定パネル**: 「ステータス分類（ON/OFF）」や「ソート（会社別/納期順）」を即座に切り替え。オフにすると単純なTODOリストとして機能。
    - **時間集計ウィジェット**: 右下に「本日の合計：製作 3.5h / 現場 2.0h」を表示。自分が今日どの工程にどのくらいリソースを割くか、一目で把握。
    - **実行**: ダッシュボードで「今日やること」を確認。ソートを切り替えて「同じ会社のものをまとめてやる」などの判断をシステムがサポート。

## 4. 会社量感カレンダー（キャパシティ可視化）
プロジェクトの工期設定時、会社全体の余力を確認するためのカレンダーです。

![会社量感カレンダー](/C:\Users\doorf\.gemini\antigravity\brain\4b9a65a8-67b4-4b6b-951c-462182743bdd\company_capacity_calendar_dark_1769642798163.png)

- **デザイン意図**:
    - **主力メンバ稼働 vs 予約済**: 各日付セルに進捗バーのようなインジケーターを表示し、「その日のキャパシティに対して何％埋まっているか（混雑度）」を直感的に表現。
    - **色による警告**: 混雑度が高い日は暖色（オレンジ〜赤）、余裕がある日は寒色（ティール〜青）で色分けし、どこに納期を設定すべきか「判断不要」に。
    - **詳細ツールチップ**: セルをホバーすると、「主力メンバ稼働合計: 24h / 予約済合計: 18h / 稼働率 75%」といった具体的な数字を表示。
    - **工期調整への活用**: プロジェクトの納期を変更する際、このカレンダーを見ながら「あ、ここは忙しいから2日後ろにしよう」といった調整をスムーズに行えます。

## 5. 操作フローの考え方
1. **新規登録**: モーダルを開き、現場情報を入力。
2. **工期設定**: 「納期」選択時にこの「会社量感カレンダー」が表示され、会社の空き状況を見ながら日付を決定。
3. **実行**: ダッシュボードでその日のタスクを消化。

これらのイメージについて、フィードバックをいただけますと幸いです。
