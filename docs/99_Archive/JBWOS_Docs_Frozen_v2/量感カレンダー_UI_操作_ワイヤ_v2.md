# 量感カレンダー UI・操作・ワイヤ（最終確定版 v2）

## 目的
量感カレンダーは、
- 納期（確定）
- 備え完了目安（ぼやけ）
- 作業量の重なり

を**文字に頼らず「気配」で把握する**ためのUIである。
判断や約束を発生させる場所ではない。

## 基本思想
- 見るだけでよい
- 操作しなくても成立する
- 未来を約束に変えない

## カレンダー構造
- 月曜始まり
- 月をまたいで**縦に連続**する
- 29,30,31,1,2 のようにシームレス表示
- 無限スクロール

## 表示要素

### 納期（確定）
- 文字で明示表示
- はっきりした色
- ドラッグ不可

### 備え完了目安（ぼやけ）
- 背景色・ぼかし・量感で表示
- 文字表示しない（既定）
- ドラッグで前後移動可能
- 約束ではない

### 背景量感
- その時期に存在する備えの総量
- 濃淡のみで表現
- 数値・警告は出さない

## ワイヤ（概念図）

```
│  29 │ 30 │ 31 │  1 │  2 │  3 │  4 │
│ ███ │ ███ │ ░░ │ ░░ │ ░░ │ ░░ │ ░░ │
│      ▲備え目安        ■納期         │

│  5 │  6 │  7 │  8 │  9 │ 10 │ 11 │
│ ░░ │ ░░ │ ░░ │ ███ │ ███ │ ░░ │ ░░ │
```

- ███：量感が高い
- ░░：量感が低い
- ■：納期（確定）

## 操作

### ドラッグ＆ドロップ
- 対象：備え完了目安のみ
- 効果：時期の再調整
- 意味：安心の再配置

### タップ
- 納期：詳細表示
- 備え：詳細表示（任意）

## 判断との関係
- 量感が高くなっても自動判断はしない
- 判断対象に出すかどうかは別UIで行う

## 禁止事項
- カレンダー上で Today を決める
- 備えを約束として扱う
- 数値化・達成率表示

## 未確定事項

- 表示期間のデフォルト範囲
