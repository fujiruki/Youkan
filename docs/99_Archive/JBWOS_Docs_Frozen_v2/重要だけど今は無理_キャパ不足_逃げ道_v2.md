# 重要だけど今は無理／キャパ不足時の逃げ道（確定版 v2）

## 目的
本ドキュメントは、
- 重要性は高い
- 納期や備え目安は近い
- しかし Today に入れられない

という **現実的なキャパ不足**を、
無理なく・罪悪感なく扱うための逃げ道を定義する。

## 基本原則
- Today は増やさない
- 無理なものは無理と扱う
- 判断を先送りしても失敗にならない

## 発生条件（例）
- Today がすでに埋まっている
- 体力・時間・集中が足りない
- 他の約束が優先されている

## UI上の選択肢（今日の約束にするか）
判断対象に対して、以下の選択肢を用意する。

- [今日の約束にする]
- [今回はしない]
- **[今は無理]**

## 「今は無理」の意味
- 今日の約束にはしない
- 失敗・先延ばしとして扱わない
- 現状のキャパを正しく反映した判断

## 遷移先
- 「今は無理」を選んだ場合：
  - 基本遷移先：備え（ぼやけ）
  - 備え完了目安が未設定の場合は設定を促す（任意）

## 表示ルール
- 「今は無理」を選んだ履歴は History に残らない
- 自己否定につながる文言は禁止

## 断る判断との関係
- 外部依頼の場合：
  - 「今は無理」は断る前段階の内部判断
- 実際に断った場合のみ History に事実として残す

## 禁止事項
- 「今は無理」を失敗扱いする
- 回数カウント
- 警告表示

## 未確定事項
- 「今は無理」の表示頻度制限
- 外注・家族共有時の見え方
