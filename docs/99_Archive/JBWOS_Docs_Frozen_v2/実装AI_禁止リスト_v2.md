# 実装AI向け 禁止リスト（これを破ったらNG）（確定版 v2）

## 目的
本ドキュメントは、実装AIが**思想を壊さないための絶対禁止事項**を列挙する。
ここに書かれている内容は、改善・最適化・親切心によって変更してはならない。

## 絶対禁止（思想破壊）

1. 未来日を約束として確定させること
2. Today 以外で約束を成立させること
3. 判断を自動化すること
4. 備え完了目安を約束扱いすること
5. 失敗・未達成を記録・表示すること
6. 優先度・ランキング・スコアを導入すること
7. Today に無制限に追加できるようにすること
8. Intent や Life をタスク化すること
9. カレンダー上で判断や約束をさせること
10. ユーザーに罪悪感を与える文言を表示すること

## 実装時の注意

- 「便利そう」「一般的だから」という理由での追加は禁止
- 自動化・最適化は必ず思想テストを通す
- 迷った場合は **何もしない** を選ぶ

## レビュー基準
- これは約束を増やしていないか？
- これは判断を増やしていないか？
- これは失敗を作っていないか？
