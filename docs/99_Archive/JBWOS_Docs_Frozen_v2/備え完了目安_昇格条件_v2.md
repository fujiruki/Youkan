# 備え完了目安 → 判断対象へ昇格する条件（確定版 v2）

## 目的
本ドキュメントは、「備え完了目安（ぼやけ）」として扱われていたものが、
**いつ・どの条件で『今日の約束にするか』の判断対象へ昇格するか**を定義する。

本昇格は自動判断ではない。
あくまで「判断対象として前に出す」ための条件定義である。

## 基本原則
- 自動で Today に入れない
- 判断は必ず人が行う
- 昇格は一時的で取り消し可能

## 昇格トリガー（いずれか）

### 1. 時間的接近
- 納期（確定）が近づいた
- 備え完了目安の時期に入った

→ 表示上、判断対象として前に出す

### 2. 重なり検知
- 複数の備え完了目安が同時期に重なっている
- 背景量感が高くなっている

→ 「判断が必要かもしれない」視覚的サインを出す

### 3. 明示操作
- ユーザーが「判断に出す」を選択
- Intent / 備えからの手動昇格

## UI上の扱い
- 昇格した項目は「今日の約束にするか」に表示される
- 自動で選択状態にはならない
- 判断しなくても問題はない

## 戻し操作
- 「今は無理」
- 「今回はしない」
→ 備え（ぼやけ）へ戻る

## 禁止事項
- 昇格＝強制判断
- 昇格＝警告
- 昇格履歴の評価

## 未確定事項
- 昇格表示の強調度
- 同時昇格数の上限
