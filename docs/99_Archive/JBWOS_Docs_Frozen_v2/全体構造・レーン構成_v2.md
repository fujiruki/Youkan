# 全体構造・レーン構成（v2）

## レーンとは
レーンとは、**アイテムの状態を表す論理的な置き場所**である。
並んで見えるが、移動は目的ではない。状態が変わるだけである。

## レーン一覧

### Inbox
- 思いついたものを即座に放り込む
- 判断はしない

### 今日の約束にするか（GDB）
- 判断対象が置かれる
- 問いは一つだけ
  - 「これは今日の約束にするか？」
- Yes → Today
- No → Intent / Life / History など

### Today
- 今日の約束
- ここに入った瞬間だけ約束が成立する
- 完了／保留／取り消しが可能

### Intent
- やれたらいいこと
- 判断を使わず一時的に前に出せる
- 期限・約束にならない

### Life
- 習慣・生活維持
- 実行できなくても失敗にならない

### History
- 事実のみを記録
- 自信の源泉として使う
