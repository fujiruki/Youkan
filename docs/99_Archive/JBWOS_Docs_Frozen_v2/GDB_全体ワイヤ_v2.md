# GDB 全体ワイヤ（判断／備え／履歴）（確定版 v2）

## 目的
本ワイヤは、GDB 周辺の **判断・備え（ぼやけ）・履歴** が
同時に俯瞰できる 1 枚構成を定義する。

UI表示名は **「今日の約束にするか」** を用いる。

## 画面構成（1枚固定・縦スクロール）

```
┌──────────────────────────────────────────┐
│ 今日の約束にするか                        │
├──────────────────────────────────────────┤
│ 【判断対象】                              │
│ ・◯◯建具 製作                            │
│ ・銀行に行く                              │
│ ・書類をまとめる                          │
│                                           │
│──────────────────────────────────────────│
│ 【備え（ぼやけ）】                        │
│ ・◯◯建具（備え完了目安：来月中旬）        │
│ ・△△案件（備え完了目安：未設定）          │
│  ※淡色・背景量感で表示                    │
│                                           │
│──────────────────────────────────────────│
│ 【履歴（参照）】                          │
│ ・××案件｜今日は約束にしない              │
│ ・■■案件｜今日の約束にした（完了）        │
└──────────────────────────────────────────┘
```

## レーン定義
- 判断対象：Today に入れるかどうかを決める唯一の場所
- 備え（ぼやけ）：約束ではない安心目安
- 履歴：事実のみの参照領域

## 表示ルール
- 判断対象のみが強調表示
- 備えは淡色・ぼかし
- 履歴は最下段・操作不可

## 操作
- 判断対象：
  - [今日の約束にする]
  - [今回はしない]
- 備え：
  - 編集（目安変更）のみ
- 履歴：
  - 閲覧のみ

## 禁止事項
- 備えから Today への直接遷移
- 履歴の編集
- 判断対象の並び替え

## 未確定事項
- 備えレーンの折りたたみ可否
