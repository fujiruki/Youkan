# 01_Project_Philosophy: 製造業と夢の両立

## 中核となる思想
このプロジェクト「JWCAD建具表作成Web（仮）」は、単なる業務効率化ツールではありません。
**「職人としての誇り高い仕事」と「個人的な夢や人間としての休息」を両立させ、精神的な充足感を得るためのパートナーシステム**です。

### 1. プロダクトの存在意義
- **既存の不満**: 一般的なカレンダーやToDoリストは「タスクをこなすこと」に主眼があり、「ものづくり」の工程（数日にわたる製作、現場作業、準備）や「心の余裕」を表現できていない。
- **目指す姿**: 
    - 複数のプロジェクト（仕事、趣味、人生のイベント）を俯瞰できる。
    - 「忙しい時期」と「暇な時期」をぼんやりと可視化し、**「あ、この時期なら夢のための作品作りに没頭できるぞ」**という気づきを与える。
    - 稼ぐべき金額（粗利）と、夢に費やす時間のバランスを計算し、**「夢を追っても食っていける」という確信と覚悟**をユーザーに与える。

### 2. "Raw Voice" からの重要キーワード
ユーザー（ドア枠さん）の言葉から抽出された、開発の憲法です。

- **「ぼんやりとした予定」**: ガチガチに固めるのではなく、緩やかに期間を確保する。
- **「な～んにもしない日」**: 効率化の逆。意図的に何もしない時間を確保し、それをシステムが肯定する機能（プラグイン）が必要。
- **「製作目安日数」**: 製造業らしさを出しつつ、ハンドメイド作家や個人のプロジェクトにも通じる汎用的な表現を模索中。
- **「見積もりとスケジュールの統合」**: 「この仕事を請けたらスケジュールがどうなるか」「いくら儲かるか」が即座に連動すること。

### 3. 対象ユーザー像
- **建具屋**: ドア、障子、家具を作る職人。
- **個人事業主の職人**: 鉄工所、家具屋、ハンドメイド作家。
- **特徴**: 事務作業よりも現場や製作が好き。だけど事務もやらなきゃいけない。夢がある。

### 4. 開発者へのメッセージ
機能を追加するときは常に自問してください。
**「それはユーザーの生産性を上げるだけでなく、ユーザーの心を豊かにするか？」**
ただの効率化ツールに成り下がらないよう、ユーザーの「遊び心」や「逃げ場」も許容する懐の深いシステムを目指してください。
