# 将来の拡張仕様: レイヤーグループベースのスケーリング

**作成日**: 2026-01-06  
**ステータス**: 将来実装予定（優先度: 中）

---

## 概要

JWCADのレイヤーグループごとの縮小倍率機能を活用し、建具のサイズに応じて最適な倍率グループに振り分けることで、各建具をセル内に最大限表示しつつ、JWCAD上での視認性を向上させる。

---

## 現状の実装 (v1.0)

**動作**:
- すべての建具に共通スケールを適用
- 最大建具を基準にスケール計算
- 相対的なサイズ比は保持される

**問題点**:
- 400mm幅の建具と1000mm幅の建具が同じシートに混在する場合
- 小さい建具は非常に小さく描画され、JWCADで見にくい可能性

---

## 提案する拡張

### レイヤーグループと倍率の対応

```
グループ1 (8-x): 1/10 倍率 → 小窓・換気窓用 (200-500mm)
グループ2 (7-x): 1/20 倍率 → 一般建具用 (500-1000mm)
グループ3 (6-x): 1/50 倍率 → 大型建具用 (1000-2000mm)
グループ4 (5-x): 1/100 倍率 → 特大建具用 (2000mm以上)
```

### 実装イメージ

```typescript
function selectLayerGroupAndScale(doorWidth: number): { group: string, scale: number } {
    if (doorWidth <= 500) {
        return { group: '8', scale: 0.1 };   // 1/10
    } else if (doorWidth <= 1000) {
        return { group: '7', scale: 0.05 };  // 1/20
    } else if (doorWidth <= 2000) {
        return { group: '6', scale: 0.02 };  // 1/50
    } else {
        return { group: '5', scale: 0.01 };  // 1/100
    }
}
```

### メリット・デメリット

**メリット**:
- 各建具がセルいっぱいに描画され視認性向上
- JWCADの標準機能を活用

**デメリット**:
- 異なる倍率グループ間でサイズ比較ができない
- レイヤー管理が複雑化
