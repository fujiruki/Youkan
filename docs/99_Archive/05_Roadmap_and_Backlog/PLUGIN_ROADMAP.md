# プラグインロードマップ

## 概要

JBWOSは拡張可能なプラグインアーキテクチャを採用し、業種や用途に応じた機能拡張を可能にする。

---

## プラグイン一覧

### 1. 製造業プラグイン（Manufacturing Plugin）
**ステータス**: 設計中

**機能**:
- プロジェクト作成時に顧客名・現場名・目標粗利を追加
- 工程ステータス管理（図面作成中→製造中→納品待ち）
- 進捗率バー表示
- 目標粗利 vs 実績のコスト管理
- 納品日カウントダウン

**影響範囲**:
- プロジェクト作成ダイアログ
- プロジェクト一覧（仕事タブ）
- プロジェクト詳細画面

---

### 2. 顧客管理プラグイン（Customer Management Plugin）
**ステータス**: 初期実装済み

**機能**:
- 顧客マスタ管理
- 顧客情報のプロジェクト紐付け
- 顧客別プロジェクト履歴

**影響範囲**:
- ハンバーガーメニュー（顧客管理リンク）
- プロジェクト作成ダイアログ（顧客選択）

---

### 3. 建具プラグイン（Tategu Plugin）
**ステータス**: 将来計画

**機能**:
- 建具設計データ管理
- DXF/JWW出力連携
- 材料リストとの連携
- プレビュー機能

**影響範囲**:
- プロジェクト詳細画面
- 材料リストプラグインとの連携

---

### 4. 見積もり・請求プラグイン（Estimation & Invoice Plugin）
**ステータス**: 将来計画

**機能**:
- 見積書作成
- 請求書作成
- PDF出力
- 顧客管理プラグインとの連携
- 入金管理

**影響範囲**:
- プロジェクト詳細画面
- 顧客管理プラグインとの連携

---

### 5. 材料リストプラグイン（Material List Plugin）
**ステータス**: 将来計画

**機能**:
- 材料マスタ管理
- プロジェクト別材料リスト
- 発注リスト生成
- 在庫管理（オプション）

**影響範囲**:
- プロジェクト詳細画面
- 建具プラグインとの連携

---

## プラグインアーキテクチャ

```typescript
interface Plugin {
  id: string;
  name: string;
  version: string;
  
  // テナント（会社）ごとに有効/無効を設定
  isEnabledForTenant(tenantId: string): boolean;
  
  // UI拡張ポイント
  extendProjectCreationDialog?: (context: DialogContext) => ReactNode;
  extendProjectListItem?: (project: Item) => ReactNode;
  extendProjectDetail?: (project: Item) => ReactNode;
  
  // データ拡張
  extendItemSchema?: () => SchemaExtension;
}
```

---

## 実装優先順位

1. **製造業プラグイン** - 現在設計中、最優先
2. **顧客管理プラグイン** - 初期実装済み、拡張予定
3. **見積もり・請求プラグイン** - 顧客管理の次
4. **材料リストプラグイン** - 建具プラグインと同時
5. **建具プラグイン** - 専門機能、最後
