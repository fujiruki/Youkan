# ユーザーモード vs 会社モード 機能定義

このドキュメントは、JBWOSにおける「個人利用（ユーザーモード）」と「会社管理（会社モード）」の機能と役割の違いを定義する。

---

## 1. 個人ユーザーモード (Personal Mode)
**「プレイヤー」としての視点。**
自分の人生と仕事を前に進めるためのすべての機能が集約される。

### コンセプト
*   **私利私欲と貢献の統合**: 自分の用事も会社の仕事も、自分のリソースを使う点では等価である。
*   **プライバシー保護**: 自分の内面（Inbox/Memo）は誰にも見せない。

### できること（Features）

#### A. タスク・プロジェクト実行
*   **Unified Dashboard (Today)**:
    *   個人のタスク（洗濯、勉強）と会社のタスク（A邸現調）が混ざったリストを消化する。
    *   「今やるべきこと」に集中する。
*   **Project Lists**:
    *   **個人プロジェクト**: 自分だけが見えるプロジェクトを作成・実行。
    *   **会社プロジェクト**: 所属する会社から割り当てられた、また自分で立案したプロジェクトを実行。
*   **Inbox / Side Memo**:
    *   脳内のモヤモヤを吐き出す。これは完全にプライベート。

#### B. 会社プロジェクトの閲覧・実行
*   **Project Detail View**:
    *   参加しているプロジェクトの**すべてのタスク**（自分以外も含む）を閲覧可能。
    *   **Task Grabbing (タスク奪取)**: 「未割り当て」のタスクを自分で取ってくる（Assign to Me）ことができる。
    *   チーム全体の進捗を把握しながら、自分の仕事を調整する。

#### C. スケジュール調整
*   **自分だけの量感カレンダー**:
    *   自分の全ての予定（公私混同）を見て、無理のない計画を立てる。
    *   「土日は休み」「平日の夜は勉強」などの個人のリズムを守る。

#### C. 設定 (Personal Settings)
*   **所属管理**: どの会社に参加するか、招待を受けるか。
*   **通知設定**: 何を通知するか。

---

## 2. 会社モード (Company Mode / Admin View)
**「マネージャー・経営者」としての視点。**
組織全体のリソースを最適化し、利益を生み出すための機能。
※「会社としてログイン」＝「その会社の管理者権限でダッシュボードを開く」ことと同義。

### コンセプト
*   **リソースの最適配分**: 誰が空いているか、誰がオーバーワークかを知る。
*   **事業の成長**: 案件を増やし、単価を上げ、効率化する。

### できること（Features）

#### A. 組織管理 (Organization)
*   **メンバー管理**:
    *   社員の追加・招待・除外。
    *   権限設定（管理者、一般、ゲスト）。
*   **機能管理 (Plugins)**:
    *   「製造業プラグイン」「顧客管理プラグイン」などの購入・ON/OFF。
    *   会社全体の支払い情報の管理。

#### B. 全体俯瞰 (Overview)
*   **Master Project List**:
    *   会社内の全プロジェクトの進捗・利益率・納期リスクを一覧表示。
    *   停滞しているプロジェクトへの介入（テコ入れ）。
*   **Team Load Map (チーム量感)**:
    *   社員全員の忙しさをヒートマップで確認。
    *   **重要**: 個人の予定は「Busy」としてマスクされるが、**「この人は来週パンパンだ」**ということはわかる。これを見て新規案件の受注可否を判断する。

#### C. プロジェクトアサイン (Assign)
*   **タスクの強制割当**:
    *   上流から落ちてきたタスクを、適切なメンバーのInboxに「依頼」として放り込む。
    *   担当者の変更、リソースの再配分。

---

## まとめ：画面の使い分けイメージ

| 機能 | 個人モード (User) | 会社モード (Company Admin) |
| :--- | :--- | :--- |
| **主な目的** | **「今日を生き抜く、タスクを終わらせる」** | **「来月を生き抜く、利益を確保する」** |
| **見える範囲** | 自分 + 参加しているPJの範囲 | 会社内のすべて (個人領域を除く) |
| **カレンダー** | 自分の予定（詳細） | メンバーの負荷状況（マスク済み） |
| **Inbox** | 自分の脳内整理 | (なし、または会社への問い合わせ箱) |
| **設定** | プロフィール、所属 | 支払い、メンバー、プラグイン |

**基本的には「個人モード」で9割の時間を過ごし、経営判断やメンバー管理をする時だけ「会社モード」に切り替える（または管理者用ダッシュボードを開く）運用となる。**
