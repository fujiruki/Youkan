# Requests

- **【最優先】** 藤田建具店OS向け 担当者割り振り・AI配車ダッシュボード
  - 背景: Youkanは個人タスク管理として非常に優秀になってきたが、藤田建具店では"誰が・いつ・何をやるか"を会社全体で管理する必要がある。
  - Beaverで案件作成→Youkanでタスク分解→AIが担当者へ割り振り→社員へ共有、という流れを標準化したい。
  - AIは決定者ではなく提案者。代表が承認して確定する。
  - 必要画面:
    1. 担当者ビュー（社員ごとに今日・明日・今週のタスク一覧）
    2. 担当者別キャパシティ（割当時間/上限時間、超過警告）
    3. AI割り振り提案画面（担当変更候補・理由・工数を提示）
    4. 社員向けToday画面（今日やる順番だけを表示）
    5. 管理者向けドラッグ&ドロップ担当変更画面
  - AIが考慮する要素:
    - estimated_minutes
    - work_days
    - assigned_to
    - 日別キャパシティ
    - スキルレベル
    - 案件の優先度
    - 納期
    - 現場場所
    - 同日にまとめられる現場
    - 外注状況
  - 将来的にはGoogleカレンダー・Beaver・Access・番頭と連携し、会社全体の司令塔として動作する。
