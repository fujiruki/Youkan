# Requests

- 【大型機能】フローView（5つ目のView）: タスク依存関係をフローチャート風に可視化・編集
  - アイテム同士に順序（依存関係）を持たせる。分岐も可能
  - @xyflow/react を使用
  - 上から下に時間軸が流れるレイアウト
  - プロジェクト共通のアイテムは背景色・大枠でグループ化。グループ同士も順序付け可能
  - 右上に未配置アイテムの最前面固定リスト。ここからドラッグ&ドロップで配置可能
  - アイテムAの上にBを重ねると「AのあとB」のリンク構造が作られる（位置は移動しない）
  - 末端でEnter → 新規アイテム追加（テキストエリアにフォーカス）
  - ガントチャートの日程移動が依存関係を飛び越えられない制約
  - Phase順: 1.データモデル → 2.フローチャートView → 3.グループ化・操作洗練 → 4.ガント連動
