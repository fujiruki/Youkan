
# 全体一覧２
- [ ] ダッシュボードの「全体一覧２」画面を新たにつくります。
・新聞風表示
・左上は現状のQuickInputです。
・QuickInput以降の並びは「今日やる」などのアイテムの状態ごとではありません。
・まずはプロジェクトに属していないアイテムたちの列挙（登録順）。それ移行は会社のプロジェクトごと、次に個人のプロジェクトごとに並びます。
・is_project=trueなアイテムがあれば、その子アイテムが1文字インデントされてその下に続きます。これがプロジェクトの見出しのようにも機能します。
・納期が近いものほど文字色が目立つように
・今日やる　などの状態はバッジで表現します。
・アイテムの右クリックメニューから「今日やる」「完了」「アーカイブ」「削除」などがあります。
 - [ ] ダッシュボードの「全体一覧２」画面において、スライドバーが２つ、フォントサイズを指定するスライダーと、画面内に収めるカラム数（1-7）を指定するスライダーが欲しい。フォントサイズは8pxから16pxの間で指定できるようにしてほしい。カラム数は1から7の間で指定できるようにしてほしい。
- [ ] ダッシュボードの「全体一覧２」画面は、新聞風に左上から順番に埋めていくデザイン。1カラムがいっぱいになったら、次のカラムにセクションが続く。この画面ではぜったいに上下スクロールはさせない。どうしても画面がいっぱいになったら、指定のカラム数よりもカラム数が多くなってもいいから、右にカラムを追加し、画面外に増やす。そのときは左右スクロール可能とする。
・フォントサイズが変わるとアイテムの高さが変わるので、１カラムに収まるアイテム数が変わります。そのときも縦スクロールがでないように、アイテムが隣の行に移動して対応してほしい。
・ItemCardコンポーネントが使えると楽だが、「今日やる」などの状態を示すバッジを表示したいという部分はこれまでのダッシュボードにはない機能だったので、どうしますかね？
・この画面には削除・アーカイブされていない全てのアイテムが表示されます。そして、完了しているものは取り消し線で表現したい。
・余白などは最小にした、一覧性の高い画面にしたい。


- [ ] 全体表示２の中でも、アイテムを左クリックしたら詳細画面が開くようにしたい。右クリックメニューも「登録と集中」と共通でいいです。プロジェクト化・今日やる（Focus)・とりかかる・保留・待ち・いつかやる・アーカイブ・削除を選べるメニューにしたい
