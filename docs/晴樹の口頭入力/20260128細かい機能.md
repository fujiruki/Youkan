# アイテムの詳細画面にて、次の挙動を直してほしい
・タスク名を編集してEnterを押しても更新されない。
・プロジェクト化したアイテムの場合は右下にサブタスク追加欄がある。しかし右下のサブタスクを記入してEnterを押しても＋ボタンを押しても反応がない。

# ダッシュボードにて
・アイテムが2行になっている。アイテムの情報は1行に収めてほしい
・アイテムの上下外側のmarginと内側のpaddingを今の10分の1にしてほしい。一覧性を優先したいので余白を削りたい。
・アイテムを右クリックしてメニューが表示されてほしい。めにゅーにはプロジェクト化・削除がある。メニューを表示してDelキーで削除ができるようにしてほしい。
・間違えて操作をしたときのために、Ctrl+Zで操作を戻したい。間違ってInboxに戻すを押したとき。間違って完了ボタンを押したときにもとに戻したい

# パノラマビュー
パノラマビューを復活させてほしい。どこから表示したら良いか、どのようにパノラマビューに切り替えるべきか議論して決めて。

#  個人設定画面
・開くとAPIエラーが右上に表示される。原因を調査し根本的に、論理的に解決して。
・表示名には、ハンバーガーメニューの上の名前の部分の情報が表示されるべきなのに、いまは空欄になっている。

# メニューからプロジェクト一覧を開き、右上の新規プロジェクトボタンを押して、プロジェクトを登録しようとしても、右上にAPI通信エラーが表示される。

# ダッシュボード
下に向かって見切れている。上下方向にスクロールできるようにしてほしい。
ただし、左右方向のスクロールは絶対なしにして。一覧性が下がるから。スマホ画面のサイズになっても、左右方向のスクロールが必要ないような幅にして。

# ダッシュボードのAlt+D
以前はこの機能があったんです。Alt+Dを押すと、直前にInboxへ放り込んだアイテムか、もしくは直前にクリックしたアイテムの詳細画面が表示される機能

# ダッシュボードのアイテム右クリック
アイテムを右クリックしたら、メニューが表示されてほしい。メニューにはプロジェクト化・削除がある。メニューを表示してDelキーで削除ができるようにしてほしい。


# 詳細画面にてプロジェクト化
詳細画面にはプロジェクト化する機能がありますよね。プロジェクト化するとサブタスクを追加できる。しかしこの機能が機能していない。原因を調査し、修正して。
