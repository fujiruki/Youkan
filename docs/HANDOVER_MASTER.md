# 次世代AI向け マスター引き継ぎ資料: Youkan (羊羹)

## 1. プロジェクトの魂 (Youkan Philosophy)
このシステムは、単なる「タスク管理」ではありません。ユーザー（春樹）が、自分の1日を **「羊羹（ようかん）を切り分けるように」**、罪悪感なく、瑞々しく管理するための **「生活管理OS」** です。

*   **「開始ボタン」はいらない**: 始めた瞬間に「今やっている」ことがわかればいい。
*   **「納期」より「My期限」**: 他人との約束（タスク）を、自分の人生のリズム（量感）に落とし込む。
*   **「公私同一」**: 仕事も遊びも同じ1日。同じカレンダーで管理する。

## 2. 現在のステータス (2026-02-09 時点)
**Youkan v1.2 Stable**: 基本機能（認証、Stream、Panolama、Newspaper、量感カレンダー）は非常に安定しています。

### 最近の大きな改善点:
- **量感カレンダーの完成**: 逆算パズル（負荷の可視化）、指示線（納期との接続）、無限スクロールが実装されました。
- **プロジェクト連動の安定化**: ヘッダーの切り替えと画面表示が1クリックで同期するようになり、UI/UXのストレスが大幅に軽減されました。
- **モチベーター設定**: ユーザーが独自の格言を登録・表示できるようになりました。

## 3. 技術スタック & 開発ルール
### コア技術
- **Frontend**: React (Vite) + TypeScript + Vanilla CSS/Tailwind.
- **Backend**: PHP 8 (API) + SQLite3 (DB).
- **Deployment**: `.\upload.ps1` (ビルドからアップロードまで自動化).

### 重要なファイルとディレクトリ
- `docs/SPEC/`: **公式仕様書 (Source of Truth)**. 開発前に必ずここを読み、変更時はここを先に直してください。
- `docs/01_RULES/Smart_Verification.md`: 自律検証プロトコル。
- `JWCADTategu.Web/src/features/core/jbwos/`: Youkanのメインロジック。
- `.agent/workflows/`: `/deploy`, `/commit` 等のAI専用コマンド。

## 4. 次のAIへ託す課題 (Next Backlog)
1.  **モバイル最適化**: 目前の課題です。現在はPC/iPad向けのため、iPhone等での視認性・操作性を向上させる必要があります。
2.  **「なにもしない日」プラグイン**: ユーザーが休息を「予約」し、それをシステムが守る機能。
3.  **Tomorrow Planning**: 明日一日の過ごし方を「羊羹を切り分けるように」予習する専用UIの検討。
4.  **建具プラグインとの疎結合化**: 現在 `jbwos` と `tategu` が共存していますが、よりプラグイン的な分離を深める余地があります。
