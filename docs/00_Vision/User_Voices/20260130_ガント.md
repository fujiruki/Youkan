# 量感カレンダー　ガントチャート画面
- [ ]ガントチャート形式のとき、背景色をつけるのは、納期と、目安期間です。この、期間＋納期というところが、通常のガントチャートと異なる点ですね。これによって、「ホントは納期までに作ればいいけど、手前で完成させたい」ということを明記できる

## フィルタの「全て」「個人」「会社」にも対応する。

## 軸の説明
- [ ] 左が項目名、右がガントチャートで横軸が日付、縦軸は項目ごと、プロジェクトに入っているものは並んでインデントされた表示。
- [ ] 行の高さは最小限にし、情報の一覧性を最優先する。
- [ ] ガントチャート上のなにもないところからドラッグすると点線の四角が表示され始め、ドラッグによってできた四角に囲まれたガントチャート上の要素が選択状態になる。その状態でどれかを掴んでドラッグし、まとめて、ブロックとしてずらすことができる。
- [ ] ガントチャートでずらされたマイ期限は、そのアイテムの設定に同期される。

- [ ] 目安期間を左右にドラッグして設定できる。そのとき、マイ期限もそれに合わせて設定される。
- [ ] 納期はドラッグできない。クリックして詳細画面から変更すべき。

- [ ] アイテムの表示方法は次の切替できる
    ・プロジェクトごとにグループ化されて表示されるモード
        プロジェクトの親アイテムは見出し的に表示される。
    ・プロジェクトグループを解除して表示するモードが有る。
        プロジェクトの親アイテムは表示されない。
- [ ] アイテムの表示順は、
    1. 納期が設定されていないもの
    2. 「納期・マイ期限の設定されている方、もしくは早い方」の早い順

- [ ] 日付の区切りとなる縦罫線が表示される。
- [ ] 日曜日の背景色は薄い赤色
- [ ] フォントサイズ・1日の横幅を調整するスライダーが上部にある
- [ ] 左のアイテム名をクリックするとそのアイテムの納期の日が表示領域に入るところまで横にスクロールする
- [ ] 上部の日の見出しの横スクロールと下の内容の横スクロールは同期しているべきである。

- [ ]  日の見出しの左右スクロールにあわせて、プロジェクト名の見出しの行が左右にスクロールしているようだ。しかしそれは不要です。画面左列のガントチャートは中身がどれだけスクロールしても常に固定されていてほしい。

- [ ] ガントチャート上のスクロールホイールの上下操作で、ガントチャートの表示領域を左右にスクロールするようにしてほしい。
    ただし、左のアイテムタイトルの列でマウスホイールを動かすと、ガントチャートの表示領域を上下にスクロールするようにしてほしい。

# 上部ヘッダーの2段目の下にあるヘッダーを定義する
    ・日付が書いてある行を「ガントチャートヘッダー」と呼ぶ
## ガントチャートヘッダーの構成
    ・前の月を表示領域中央になるようスクロールするボタン
    ・現在の表示領域中央の年月表示　（スクロールすると自動的に変化）
    ・次の月を表示領域中央になるようスクロールするボタン
    ・「今月を表示」ボタン（今月が表示領域の中央になるようスクロールする）
    ・日次設定ボタン（日を選択してボタンを押すと、その日の個人・各社への時間配分を個別に設定できる。個人設定での曜日設定に特定日のみ上書きできる）

・その左が項目名、右がガントチャートで横軸が日付、縦軸は項目ごと、プロジェクトに入っているものは並んでインデントされた表示。

・ガントチャート上のなにもないところからドラッグすると点線の四角が表示され始め、ドラッグによってできた四角に囲まれたガントチャート上の要素が選択状態になる。その状態でどれかを掴んでドラッグし、まとめて、ブロックとしてずらすことができる。
・行の高さは最小限にし、情報の一覧性を最優先する。
・ガントチャートに、それぞれの日の1日のmax労働時間を超えないように調整をしていく機能がほしい。
・ガントチャートでずらされた納期や毎納期は、同期・編集される。

今月を表示ボタンは、それを押すと、今月が表示領域の中央になるようにスクロールするという意味です。

今月を表示ボタンは、それを押すと、今月が表示領域の中央になるようにスクロールするという意味です。

カレンダーが表示されている範囲のアイテムはすべて表示してください。スクロールを端っこまですると続きを読み込んでシームレスに無限スクロールできるようにしてください。もちろんその時読み込んで表示しようとするアイテムも読み込んでフィルタなど条件に応じて正しく表示してほしい
