# タイムライン画面
## この機能の概要
タイムライン画面では、行がメンバー、列が日になっており、いつこれをやるかというのを書くことができる、一直線のカレンダーになっている。メンバーAに、何日から何日はこの仕事、何日から何日はこの仕事というように仕事を振り込む。
ユーザーアカウントが個人モードで表示している場合は、自分だけが表示される。

## このタイムライン画面では、アカウントとモードだけが影響する。
フィルタは影響しない。

## タイムラインのデザインのイメージ
タイムライン上のタスクは、
[タイムラインのデザイン案](20260224_タイムライン.md)

## ドラッグドロップによるタスクの移動
タスクは目安期間が表示されている状態。
パズルのピースのようにタスクをはめ込んでいく。
タスクをドラッグドロップしてはめ込んでいく。
ドラッグ中も、この場所においたら・・・というシルエットが半透明で表示される。
アイテムにはプロジェクトの色を薄めた背景色の四角形として表示される。
目安期間が休みの日を挟む場合は、そのように分断されて表示される。

タスクは現在のモードで表示できるものだけを表示する。
例えば、個人モードなら個人のタスクだけを表示する。
会社モードなら会社のタスクだけを表示する。
