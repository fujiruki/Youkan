# 量感カレンダー　ガントチャート画面
- [ ]ガントチャート形式のとき、背景色をつけるのは、納期と、目安期間です。この、期間＋納期というところが、通常のガントチャートと異なる点ですね。これによって、「ホントは納期までに作ればいいけど、手前で完成させたい」ということを明記できる

- [ ] 目安期間を左右にドラッグして設定できる。そのとき、マイ期限もそれに合わせて設定される。
- [ ] 納期はドラッグできない。クリックして詳細画面から変更すべき。

- [ ] アイテムの表示方法は次の切替できる
    ・プロジェクトごとにグループ化されて表示されるモード
        プロジェクトの親アイテムは見出し的に表示される。
    ・プロジェクトグループを解除して表示するモードが有る。
        プロジェクトの親アイテムは表示されない。
- [ ] アイテムの表示順は、
    1. 納期が設定されていないもの
    2. 「納期・マイ期限の設定されている方、もしくは早い方」の早い順

- [ ] 日付の区切りとなる縦罫線が表示される。
- [ ] 日曜日の背景色は薄い赤色
- [ ] フォントサイズ・1日の横幅を調整するスライダーが上部にある


- [ ] 左が項目名、右がガントチャートで横軸が日付、縦軸は項目ごと、プロジェクトに入っているものは並んでインデントされた表示。
- [ ] 行の高さは最小限にし、情報の一覧性を最優先する。
- [ ] ガントチャート上のなにもないところからドラッグすると点線の四角が表示され始め、ドラッグによってできた四角に囲まれたガントチャート上の要素が選択状態になる。その状態でどれかを掴んでドラッグし、まとめて、ブロックとしてずらすことができる。
- [ ] ガントチャートでずらされたマイ期限は、そのアイテムの設定に同期される。
