私はとにかく、藤田晴樹の特性に合わせたシステムをつくりたい
・時間管理が少しでも気持ちを楽にさせつつスケジュール管理が上手にできるようにしてあげたい。
・納期遅れをなくして上げたい。
・納期ギリギリで苦しむことをなくしたい。
・受注した仕事はいいものを納品したい。
・現状のスケジュールで受注できないものを「やはり受注できない」とわかりやすくなりたい。
・時間管理のために自分の抱えてるタスクを把握したい。
・時間管理ツールを開くのが億劫になりたくない。
・呼びやすい名前（3-4文字）の時間管理ツールであってほしい
・時間管理ツールに登録が一瞬でできたい。私は登録しようとしながらすぐ忘れちゃうので、とりあえずタイトルだけでも登録。余裕があればすぐに納期などの詳細を入力する。
・「余裕がない、どんどん他のタスクを登録したいのに忘れていきそう」というときは、とりあえずタイトルだけでも次々登録できる
・登録したタスク立ちを、時間の重なりなく、無理なくスケジュールに配置するのが億劫にならないようにしたい。登録はしてあるけど、その調整が面倒で先延ばしにしてしまうことが多い。


・タスクのカードは一覧性重視のため、１行に収まるデザインにして、上下方向のpadding/marginはできるだけ小さくしてください。3pxとか。
・また、以前Today画面にあったような、1日24時間のうちのどこまで経過したかを表すプログレスバーのようなものがさりげなく表示されていてほしい。「夜ご飯まで時間がないぞ」というのに気づかせてほしいため。
