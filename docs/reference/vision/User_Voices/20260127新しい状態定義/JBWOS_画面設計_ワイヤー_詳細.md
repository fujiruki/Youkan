
# JBWOS 画面設計・ワイヤー（完全版）

## 1. 全体構成
- ホーム = Dashboard
  - Focus View（標準）
  - Panorama View（切替）

## 2. Dashboard（Focus View）
```
┌────────────────────────────┐
│ Focus (最大3件)             │
│ ────────────────────────── │
│ ▶︎ A邸 建具製作（未着手）   │
│ ▶︎ B邸 見積（着手中）      │
│ ▶︎ 請求書作成（未着手）    │
└────────────────────────────┘

┌────────────────────────────┐
│ Inbox                       │
│ ・C邸 建具                  │
│ ・道具修理（期限バッジ）   │
└────────────────────────────┘

┌────────────────────────────┐
│ Pending                     │
│ ・D邸（判断テンプレ表示） │
└────────────────────────────┘

┌────────────────────────────┐
│ Waiting                     │
│ ・金物待ち                 │
└────────────────────────────┘
```

## 3. Panorama View
```
A邸
  ├ 建具A
  ├ 建具B
B邸
  ├ 見積
請求書
道具修理
```
- 目的：忘却防止・全体把握
- 操作は最小限（スクロールのみ）

## 4. カードUIルール
- タップ = 詳細画面
- 状態変更 = スワイプ / 長押しメニュー
- 着手中表示 = 左端に細いバー

## 5. 詳細画面
- 納期 / マイ期限 / 目安日数
- 横メモ（即入力・即保存）
- 子タスク一覧

## 6. メッセージ設計
- Focusに1件入った時
  「今日はこれに向き合うと決めました」
- 未達成時
  「判断は明日更新できます」

## 7. 実装ヒント
- 状態 = Enumは最小
- 表示 = ViewModelで分岐
- 判断テンプレ = Pending移動時モーダル
