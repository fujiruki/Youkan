- [ ] ダッシュボードの「全体一覧２」画面は、新聞風に左上から順番に埋めていくデザイン。1カラムがいっぱいになったら、次のカラムにセクションが続く。この画面ではぜったいに上下スクロールはさせない。どうしても画面がいっぱいになったら、指定のカラム数よりもカラム数が多くなってもいいから、右にカラムを追加し、画面外に増やす。そのときは左右スクロール可能とする。

## 利用の流れ
- [ ] 夜、ユーザーはこの画面でどんなタスクやプロジェクトが待ち構えているかを一望できる。この状態で、「そういえば今日はこの作業をすべきだって追加があったんだった。」とタスクを追加する。

- [ ] 新しい仕事「A様リフォーム」の依頼が「デバッグ社」にあったら、Projectタブを開いて、Companyタブを開いて、新規プロジェクトボタンを押して「A様リフォーム」を「デバッグ社」のプロジェクトとして作成する。そのプロジェクトをクリックするとProjectFocusモードになると同時に、全体一覧２が開く。その中の「ここに吐き出す」欄をつかって、このプロジェクトをタスク分解し、思いつくタスクを次々登録する。登録されたアイテムは、「A様リフォーム」プロジェクトに属し、「デバッグ社」にも属している。次々登録してリストアップされたものを左クリックして詳細画面をひらき、それぞれの納期・毎納期・目安日数を登録する。すると、リストアップされたものは納期ごとに並ぶ。アイテムを右クリックして「今日やる」を押すと、FOCUS状態になり、そのアイテムの右側に青い●が表示される。いくつかFOCUS状態にしておいて、今日の活動が始まる。青い●がついているタスクを実行し、完了したらそのアイテムを右クリックしてDone状態にする。すると取り消し線が表示され、青い●が消える。青い●が画面上からすべて消えたら、今日のタスクが完了したということになる。


## それぞれの状態と表示方法
- プロジェクト
	プロジェクトの先頭には目立たないフォルダーアイコンを置く。プロジェクトの右側に＋ボタンがあり、それを押すと、その下に子アイテムを追加できる入力欄が表示される。子アイテムが登録されると、そのアイテムはプロジェクトに属する。子アイテムの右側にも＋ボタンがあり、それを押すと、その下に孫アイテムを追加できる入力欄が表示される。孫アイテムが登録されると、そのアイテムはプロジェクトに属する。
- 保留（Pending)
	フォント文字：黒
	●の表示：なし
- 待ち（Waiting)
	フォント文字：グレー
	●の表示：グレー
- 今日やる（FOCUS)
	フォント文字：黒
	●の表示：タイトル文字右側に左寄せで青色
- 現在取り掛かり中（Execute)
	フォント文字：黒
	●の表示：緑に点灯した●
- 完了（Done)
	フォント文字：グレー＋取り消し線
	●の表示：なし
- いつかやる（Someday)
	フォント文字：黒
	●の表示：なし
	全てのアイテムの中で一番右下に表示する。
