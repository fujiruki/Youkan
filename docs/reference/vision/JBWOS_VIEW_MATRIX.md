# JBWOS 視点マトリックス：現実の切り取り方 (Vision)

## 思想的背景
JBWOSは、一人の人間が持つ複数の役割（個人、経営者、社員）を、状況に応じてシームレスかつ厳格に切り替えるためのOSである。
量感カレンダーにおける「視点の切り替え」は、単なるフィルタリングではなく、その瞬間の「自分の現実」の再定義である。

## 視点マトリックス (View Matrix)

| ログインアカウント | 表示モード | 表示対象（分子：タスク） | 計算キャパ（分母：能力） | 思想的意味 |
| :--- | :--- | :--- | :--- | :--- |
| **ユーザー(個人)** | **個人モード** | 本人の全タスク（担当or作成） | 生活キャパ ＋ 所属全社の稼働時間 | **「全人的な人生」の俯瞰**。どこにいても自分の24時間は1つであることを直視する。 |
| **ユーザー(個人)** | **会社モード** | その会社の自分の担当分のみ | 自分が「その会社」に割いている時間 | **「プロとしての集中」**。他社のノイズを消し、今向き合うべき組織での役割に没頭する。 |
| **会社(法人)** | **社内モード** | その会社に属する全タスク | 主力メンバー全員の当該会社用キャパ合計 | **「組織の稼働管理」**。チーム全体のリソースと負荷を鳥瞰し、経営判断を行う。 |
| **会社(法人)** | **会社モード** | その会社に属する全タスク | 主力メンバー全員の当該会社用キャパ合計 | 社内モードと同様（法人格としての統一視点）。 |

## 権限・可視性の原則 (Visibility Principle)
- **所有権の尊重**: 自分が直接関与（作成または担当）している事実は、コンテキストに関わらず「真実」として常にマスクせず表示する。
- **組織の境界**: 自分が所属する組織のタスクは、詳細を確認できる権利を持つ。
- **プライバシーの保護**: 全く関与のない他者のタスクは、単なる「負荷の塊（Busy）」としてのみ認識し、中身を問わない。
