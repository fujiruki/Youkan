## 1. 基本計算式

### 1.1 原価計算の基本式

建具の材料費は、加工・製材時のロスを含めた「木取り寸法」に基づく**材積（m³）**から算出します。

$$
\text{原価} = \text{材積}(m^3) \times \text{立米単価}(円/m^3) \times \text{本数}
$$

$$
\text{材積}(m^3) = \frac{(W + W_{margin}) \times (T + T_{margin}) \times (L + L_{margin} + Hozo \times 2)}{10^9}
$$

**単位**:
- 寸法: **mm**
- 材積: **m³**
- 立米単価: **円/m³**

---

### 1.2 デフォルト設定値

| 項目 | 記号 | デフォルト値 |
|:---|:---|:---|
| 立米単価 | `pricePerM3` | **800,000円/m³** |
| 幅余裕 | `W_margin` | 5mm |
| 長さ余裕 | `L_margin` | 50mm |
| 厚さ余裕 | `T_margin` | 3mm |
| ホゾ長さ（片側） | `hozoLength` | 30mm |
| マークアップ率 | `markup` | 20% |
| 消費税率 | `taxRate` | 10% |

## 2. 用語定義

### 2.1 寸法の種類

| 用語 | 説明 | 例 |
|:---|:---|:---|
| **見付（みつけ）** | 正面から見える幅寸法 | 框の幅: 100mm |
| **見込み（みこみ）** | 奥行き方向の厚さ | 建具の厚み: 30mm |
| **仕上がり長さ** | 完成時の可視部分の長さ | 上桟の内法長さ: 600mm |
| **木取り寸法** | 加工前の材料取り寸法（余裕＋ホゾ含む） | 105 × 33 × 850mm |

### 2.2 ホゾ（Hozo）

**ホゾ**: 部材同士を接合するための凸形状。材料長さに加算される。

- **片側**: 30mm（デフォルト）
- **両側**: 60mm（片側×2）

**部材別ホゾ長さ**:

| 部材 | ホゾ長さ（片側） | 理由 |
|:---|:---|:---|
| 縦框 | 0mm | 受け側（メス）のため不要 |
| 上桟・下桟・中桟 | 縦框幅 | 縦框に食い込む |
| 組子ヨコ | 縦框幅 | 同上 |
| 束・組子タテ | 30mm | 設定値 |

## 3. 部材別詳細仕様

### 3.1 縦框 (Stile)

**役割**: 建具の左右を構成するメインの柱。

**作図範囲**:
- 左縦框: `(0, 0)` ～ `(Sw, DH)`
- 右縦框: `(DW - Sw, 0)` ～ `(DW, DH)`

**積算仕様**:
```
本数: 2本
見付: Sw (例: 100mm)
見込み: D (例: 30mm)
仕上がり長さ: DH (例: 2000mm)
ホゾ: 0mm
```

**木取り寸法**:
```
(Sw + 5) × (D + 3) × (DH + 0 + 50)
例: (100+5) × (30+3) × (2000+0+50) = 105 × 33 × 2050 mm
```

---

### 3.2 上桟 (Top Rail)

**役割**: 建具最上部の横枠。

**作図範囲**:
- `(Sw, DH - Trw)` ～ `(DW - Sw, DH)`

**積算仕様**:
```
本数: 1本
見付: Trw (例: 100mm)
見込み: D (例: 30mm)
仕上がり長さ: DW - (Sw × 2) (例: 600mm)
ホゾ（片側）: Sw (例: 100mm)
```

**木取り寸法**:
```
(Trw + 5) × (D + 3) × (仕上がり長さ + Sw×2 + 50)
例: (100+5) × (30+3) × (600+200+50) = 105 × 33 × 850 mm
```

---

### 3.3 下桟 (Bottom Rail)

上桟と同様の計算。見付を`Brw`に読み替える。

---

### 3.4 中桟 (Middle Rail)

**役割**: 構造補強のための中間の横桟。

**積算仕様**:
```
本数: 指定本数 (例: 1本)
見付: Mrw (例: 30mm)
見込み: D (例: 30mm)
仕上がり長さ: DW - (Sw × 2) (例: 600mm)
ホゾ（片側）: Sw (例: 100mm)
```

**位置指定**:
- `middleRailPosition`が設定されている場合: その高さに配置
- 未設定の場合: 内法を等分割

---

### 3.5 束 (Tsuka / Vertical Mullion)

**役割**: 中桟と下桟の間に入れる短い縦材。

**条件**: **中桟が1本以上存在する場合のみ計算**

**積算仕様**:
```
本数: 指定本数 (例: 3本)
見付: TsukaWidth (例: 30mm)
見込み: D (例: 30mm)
仕上がり長さ: 中桟間スペース高さ (自動計算)
ホゾ（片側）: 30mm
```

**長さ計算ロジック**:
```
内法高さ = DH - Trw - Brw
中桟合計高さ = Mrw × 中桟本数
残余スペース = 内法高さ - 中桟合計高さ
スペース数 = 中桟本数 + 1
1スペース高さ = 残余スペース ÷ スペース数
束の長さ = 1スペース高さ (floor)
```

---

### 3.6 組子タテ (Kumiko Vertical)

**積算仕様**:
```
本数: 指定本数
見付: KvWidth (例: 6mm)
見込み: D (例: 30mm)
仕上がり長さ: 
  - 中桟なし → 内法高さ全体
  - 中桟あり → 上桟下端 ～ 最上部中桟上端 (≒ 1スペース高さ)
ホゾ（片側）: 30mm
```

---

### 3.7 組子ヨコ (Kumiko Horizontal)

**積算仕様**:
```
本数: 指定本数
見付: KhWidth (例: 6mm)
見込み: D (例: 30mm)
仕上がり長さ: DW - (Sw × 2)
ホゾ（片側）: Sw
```

## 4. 計算例（実践）

### 4.1 建具仕様

```
建具サイズ: W800 × H2000 × D30mm
縦框: 100mm
上桟: 100mm
下桟: 200mm
中桟: 1本 (見付30mm)
束: 3本 (見付30mm)
組子タテ: 5本 (見付6mm)
組子ヨコ: 3本 (見付6mm)
```

**設定値**:
```
立米単価: 800,000円/m³
幅余裕: 5mm
長さ余裕: 50mm
厚さ余裕: 3mm
```

---

### 4.2 部材別計算

#### **（1）縦框** - 2本

```
見付: 100mm, 見込み: 30mm, 長さ: 2000mm
ホゾ: 0mm

木取り寸法:
W = 100 + 5 = 105mm
T = 30 + 3 = 33mm
L = 2000 + 0 + 50 = 2050mm

材積 (1本):
Vol = 105 × 33 × 2050 ÷ 10^9 = 0.00709 m³

材積 (2本):
Vol_total = 0.00709 × 2 = 0.01418 m³

原価:
Cost = 0.01418 × 800,000 = 11,344円
```

---

#### **（2）上桟** - 1本

```
見付: 100mm, 見込み: 30mm
仕上がり長さ: 800 - (100×2) = 600mm
ホゾ（片側）: 100mm → 両端で200mm

木取り寸法:
W = 100 + 5 = 105mm
T = 30 + 3 = 33mm
L = 600 + 200 + 50 = 850mm

材積:
Vol = 105 × 33 × 850 ÷ 10^9 = 0.00294 m³

原価:
Cost = 0.00294 × 800,000 = 2,352円
```

---

#### **（3）下桟** - 1本

```
見付: 200mm, 見込み: 30mm
仕上がり長さ: 600mm
ホゾ: 200mm

木取り寸法:
W = 200 + 5 = 205mm
T = 30 + 3 = 33mm
L = 600 + 200 + 50 = 850mm

材積:
Vol = 205 × 33 × 850 ÷ 10^9 = 0.00575 m³

原価:
Cost = 0.00575 × 800,000 = 4,600円
```

---

#### **（4）中桟** - 1本

```
見付: 30mm, 見込み: 30mm
仕上がり長さ: 600mm
ホゾ: 200mm

木取り寸法:
W = 30 + 5 = 35mm
T = 30 + 3 = 33mm
L = 600 + 200 + 50 = 850mm

材積:
Vol = 35 × 33 × 850 ÷ 10^9 = 0.00098 m³

原価:
Cost = 0.00098 × 800,000 = 784円
```

---

#### **（5）束** - 3本

```
# スペース高さ計算
内法高さ = 2000 - 100 - 200 = 1700mm
中桟合計高さ = 30 × 1 = 30mm
残余スペース = 1700 - 30 = 1670mm
スペース数 = 1 + 1 = 2
スペース高さ = 1670 ÷ 2 = 835mm

見付: 30mm, 見込み: 30mm, 長さ: 835mm
ホゾ（片側）: 30mm → 両端で60mm

木取り寸法:
W = 30 + 5 = 35mm
T = 30 + 3 = 33mm
L = 835 + 60 + 50 = 945mm

材積 (1本):
Vol = 35 × 33 × 945 ÷ 10^9 = 0.00109 m³

材積 (3本):
Vol_total = 0.00109 × 3 = 0.00327 m³

原価:
Cost = 0.00327 × 800,000 = 2,616円
```

---

#### **（6）組子タテ** - 5本

```
長さ = スペース高さ = 835mm
見付: 6mm, 見込み: 30mm
ホゾ: 60mm

木取り寸法:
W = 6 + 5 = 11mm
T = 30 + 3 = 33mm
L = 835 + 60 + 50 = 945mm

材積 (5本):
Vol = 11 × 33 × 945 × 5 ÷ 10^9 = 0.00172 m³

原価:
Cost = 0.00172 × 800,000 = 1,376円
```

---

#### **（7）組子ヨコ** - 3本

```
見付: 6mm, 見込み: 30mm
仕上がり長さ: 600mm
ホゾ: 200mm

木取り寸法:
W = 6 + 5 = 11mm
T = 30 + 3 = 33mm
L = 600 + 200 + 50 = 850mm

材積 (3本):
Vol = 11 × 33 × 850 × 3 ÷ 10^9 = 0.00093 m³

原価:
Cost = 0.00093 × 800,000 = 744円
```

---

### 4.3 合計原価

```
縦框:        11,344円
上桟:         2,352円
下桟:         4,600円
中桟:           784円
束:           2,616円
組子タテ:     1,376円
組子ヨコ:       744円
─────────────────
材料費合計:  23,816円

マークアップ (20%):
単価 = 23,816 × 1.2 = 28,579円

消費税 (10%):
税込価格 = 28,579 × 1.1 = 31,437円
```

## 5. UI表示要件

積算詳細画面（`EstimationPanel`）では、計算根拠が明確になるように以下を表示：

### 5.1 表示項目

| 列 | 内容 | 例 |
|:---|:---|:---|
| **部材名** | 部材の名称 | 縦框 |
| **仕上がり寸法** | W × D × L (mm) | 100×30×2000 |
| **ホゾ加算** | 両端合計 | +0mm |
| **本数** | 数量 | 2本 |
| **小計** | 原価 | 11,344円 |

### 5.2 検算可能性

**重要**: ユーザーが電卓で以下の計算を実行したとき、UIの金額と**完全に一致**すること。

```
検算式:
(W + W_margin) × (T + T_margin) × (L + Hozo×2 + L_margin) × 本数 ÷ 10^9 × 立米単価
```
