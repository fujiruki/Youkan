# History（事実ログ）設計（確定版 v2）

## 位置づけ
History は **実際に起きた事実のみを記録するレーン**である。
評価・反省・改善提案は行わない。

## History に入るもの
- Today で完了したもの
- Intent で「もうやらない」を選んだもの
- Life を実行して完了した事実

## History に入らないもの
- やらなかったこと
- 迷っていたこと
- 失敗と解釈されうる情報

## 表示ルール
- 時系列で並ぶ
- 色は落ち着いた単色
- 点数・達成率・連続記録は表示しない

## 操作
- 閲覧のみ
- 編集・削除はしない

## 心理設計
- History は「自信の源泉」として使う
- 見ることで「できた事実」だけが残る
- 見なくても困らない

## 禁止事項
- 反省コメントの入力
- 未達成の可視化
- 比較グラフの表示

## 未確定事項
- 保持期間
- 検索・フィルタ条件
