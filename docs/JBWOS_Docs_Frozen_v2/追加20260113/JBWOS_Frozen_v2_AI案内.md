# JBWOS Frozen v2 保管ドキュメント案内（AI向け）

## このフォルダについて
このフォルダには、JBWOS（Judgment-Based Work Operating System）Frozen v2 の設計ドキュメント一式が保管されています。

本一式は以下の条件で作成されています。

- 会話ログに基づく **忠実な確定設計**
- 解釈・改善・最適化は禁止
- 思想・用語・UI・状態遷移・ワイヤー・例外処理まで含む
- 実装AIにそのまま渡すことを前提とした粒度

本フォルダ内の内容が **唯一の正** であり、
他の資料・一般論・タスク管理理論を参照してはいけません。

---

## 設計の根本思想（超要約）

- 約束は **Todayに入った瞬間だけ**
- 判断とは「今日の約束にするか」を決めること
- 備え（Preparation）は **ぼやけていて約束ではない**
- 納期は現実、備えは安心のための目安
- 失敗・遅れ・できなかった、という概念をUIに出さない
- 人が自分を責めずに続けられることを最優先する

## AIへの絶対ルール（再掲）

- 意味を変えない
- 言葉を置き換えない
- 勝手に良くしない
- 「普通はこう」を持ち込まない
- 不明点は「未確定」と明記する

## この設計のゴール

これは効率化ツールではありません。
人が **自分との約束を守れる確率を上げるためのOS** です。

それ以外の目的を付与してはいけません。
