# 量感カレンダー：外注・父が関わる建具案件シミュレーション（Frozen v2準拠）

本ドキュメントは、**外注先・父（他者）が関与する場合**において、
JBWOS Frozen v2 の思想がどのように保たれるかを示す具体例である。

※ 判断の自動化なし  
※ 約束は Today のみ  
※ 他者都合は「備え」に留める

---

## 案件概要

- 案件名：△△邸 建具製作
- 納期（取付日）：**4/15（確定）**
- 制作所要日数（目安）：**自作5日＋外注待ち**
- 関与者：
  - 父：部材加工
  - 外注：塗装

## まとめ

- 他者都合はすべて「備え」に留まる
- 判断は必ず人が行う
- waiting_external は焦らせない
- 外部が絡んでも約束は増えない

これは **Frozen v2 が想定する複雑ケースの成功例**である。
