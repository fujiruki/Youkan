# 断る判断（外部・納期が守れない場合）（確定版 v2）

## 目的
本ドキュメントは、
- 外部の相手が存在する
- 納期・取付日などの約束が関与する
- 現実的に守れない可能性が高い

という状況において、**無理に引き受けない判断**を
システム上で自然に行えるようにすることを目的とする。

## 基本原則
- 納期を守れない仕事は引き受けない
- 無理に Today を増やさない
- 断ることを失敗・逃げと扱わない

## 発生条件
以下のいずれかを満たす場合に「断る判断」が視野に入る。

- 納期（確定）が近い
- 備え完了目安が重なっている
- Today に入れられる余地がない
- すでに「今は無理」が複数回発生している

## UI上の位置
- 「今日の約束にするか」画面で選択可能
- 通常の判断と同列に置かれる

### 選択肢
- [今日の約束にする]
- [今回はしない]
- [今は無理]
- **[断る]**

## 「断る」の意味
- 外部依頼を引き受けないと決める内部判断
- その時点で Today や備えには入らない
- 実際の連絡行為はシステム外で行う

## 遷移先
- 選択後：History
- 記録内容：
  - 「断った」という事実のみ
  - 理由・反省・評価は残さない

## 心理設計
- 断る判断は「自分を守る判断」
- キャパを正確に把握できている証拠
- 後ろめたさを生まない文言を使用

## 禁止事項
- 断った回数の表示
- 断り理由の入力強制
- 後悔を促す文言

## 未確定事項
- 断る判断を表示する条件の細かな閾値
- 家族・外注が関わる案件での共有表現
