# 用語対応表.md

※本ファイルは、このチャット終盤時点で確定した用語とUI表示を、
**言い換え・再解釈を行わず**対応表として整理したものである。

---

## 基本方針（確定）

- UIに表示される文言は **すべて日本語**
- 用語は「何を書く場所か」が直感的に分かることを優先
- 「〜なこと」という形式を基本とする
- 内部概念名とUI表示名を明確に分ける

---

## 用語対応表（確定）

| 内部概念 | UI表示名 | 説明（終盤合意どおり） |
|---|---|---|
| Inbox | 放り込む | 判断・整理前の一時置き場 |
| GDB (Global Decision Board) | 判断すること | 今日・今後について**判断が必要なこと** |
| Judgment | 進め方を決めること | 今日進めるか／まだ進めないかを決める行為 |
| Today | 今日やること | **今日だけの約束**を置く場所 |
| Execution | 今すすめていること | Todayに入った結果として進行中のこと |
| Intent | やれたらいいこと | 近々・いつか・やりたいと思っていること |
| Life | 生活のこと | 洗濯・掃除・薬などの日常・習慣 |
| 備え完了目安 | 備えておくこと | 自分が安心するための完了目安 |
| 納期 | 納期 | 相手との約束の日付 |
| History | これまでの事実 | 評価しない事実ログ |
| 量感カレンダー | 量感カレンダー | 忙しさ・負荷を**気配**で見るカレンダー |
| ついで | ついでにやること | 外出ついで・軽い行動 |
| Intent Boost | 今日だけ前に出す | 判断を使わず一時的に前に出す操作 |

---

## 補足（確定）

- 「判断」「約束」「実行」は明確に分離する
- Intent / Life / 備え完了目安 は **約束ではない**
- Today に入った瞬間のみが「約束」
