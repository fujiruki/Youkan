# 実装AIへの指示文（添え文・最終版 v2）

あなたはこれから、JBWOS の設計ドキュメント一式を読みます。

## 最重要指示（絶対）
- これは**要約でも再設計でもありません**
- あなたの役割は **忠実な実装** です
- 設計に書かれていないことを「補完」してはいけません
- 良かれと思った改善・一般的UIの適用は禁止です

## 読み方
1. **配布用索引_読み順_v2.md** を最初に読むこと
2. 指定された順番を絶対に崩さないこと
3. 未確定と書かれている箇所は、そのまま未確定で止めること

## 実装中の判断基準
迷ったら次を自問してください：
- これは約束を増やしていないか？
- これは判断回数を増やしていないか？
- これは失敗や罪悪感を生んでいないか？

一つでも「Yes」なら実装してはいけません。

## 禁止
- 設計思想をUI都合で変えること
- スケジューラ・TODOアプリ的挙動への寄せ
- 自動最適化・AI提案の先行実装

## ゴール
ユーザーが
- 約束を減らし
- 判断に疲れず
- 自信を積み上げられる

その一点だけを守って実装してください。
