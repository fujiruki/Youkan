# 量感カレンダー（Calendar B）詳細仕様書

## 1. 概要
JBWOSにおける「量感カレンダー」は、単なるスケジュール帳ではなく、人生の「密度（熱量）」を可視化し、判断疲れを軽減するための伴走ツールである。

## 2. コア機能

### 2.1 納期カードと制作期間の分離
- **納期カード**: 本来の締切日（Deadline）に、アイテム名とプロジェクト名を表示。
- **制作期間（量感）**: 「マイ期日（My Due Date）」から遡って、必要工数分だけ背景色（熱量）が表示される。

### 2.2 シーケンシャル・パッキング（逆算消費）ロジック
- **アルゴリズム**: 
    - マイ期日が早いタスクから順に、各日の「空き枠」を埋めていく。
    - 空き枠がなくなると、さらに過去の日付へと配分が押し出される。
- **効果**: 「いつから始めないと間に合わないか」という物理的現実を裁くことなく提示する。

### 2.3 多重コンテキスト・キャパシティ管理
- **個別設定**: 曜日・コンテキスト（個人/プロジェクトA/プロジェクトB）ごとに稼働可能時間を設定可能。
- **フィルタ連動**: [全表示][個人][会社A] 等のフィルタにより、計算対象（分子）とMAXキャパシティ（分母）が動的に切り替わる。

### 2.4 「静寂」の予約（Nothing Days）
- **操作**: カレンダー上での右クリックメニューから「この日を静寂にする」を選択。
- **挙動**: その日のキャパシティを強制的に 0h にし、全タスクを前後へ自動的に回避させる。

## 3. ビジュアル設計
- **色彩表現**: 
    - 100%超は警告の赤ではなく、エネルギッシュな「高熱（Intensity）」として表現。
    - コンテキストごとにベースカラー（青/温色/緑）を使い分け。
- **コネクション**: 納期カードをホバー/クリックした際、そのタスクのために色が変化している「制作期間」のセルをハイライト/線で繋ぐ。

## 4. 統合
- **詳細画面**: メインカレンダーと同じロジック・UIを持つ「ミニ量感カレンダー」を配置。フィルタ切替や、日付変更時のリアルタイムシミュレーションが可能。
