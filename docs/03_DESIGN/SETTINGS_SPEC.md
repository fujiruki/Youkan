# 設定画面（Company & Personal Settings）仕様書

**作成日:** 2026年1月25日
**Version:** 1.0

## 1. 概要
本ドキュメントは、Tategu Design Studioにおける「会社アカウント」および「個人アカウント」向けの設定画面の仕様と、その決定経緯をまとめたものである。
ユーザーの属性（Tenant所属か、個人利用か）に応じて、適切な設定項目を提供する。

## 2. 決定経緯（AI専門家会議 2026.01.25）
Mozzarella (PM)、Leonardo (UX)、Ken (Security) による協議の結果、以下の指針が決定された。

*   **会社設定 (Company Settings):** 組織管理、セキュリティ、リソース管理（Harukiモデル）を重視。
*   **個人設定 (Personal Settings):** ユーザビリティ、個人の働き方（My稼働）、アカウントセキュリティを重視。
*   **UI/UX:** ハンバーガーメニューにおいて、現在ログイン中のアカウント種別（会社名/個人）を明確に表示する（実装済み）。

## 3. Company Settings (会社設定)
テナント管理者（Owner/Admin）のみがアクセス可能。

| カテゴリ | 項目名 | 仕様・機能 | 優先度 |
| :--- | :--- | :--- | :--- |
| **基本情報** | プロフィール | ・会社名 (Name)<br>・代表者名<br>・住所、電話番号<br>・ロゴ画像アップロード | 高 |
| | インボイス設定 | ・インボイス登録番号<br>・請求書等の定型文設定 | 中 |
| **チーム管理** | メンバー一覧 | ・所属メンバーのリスト表示<br>・招待（Email招待）<br>・権限変更（管理者/一般）<br>・メンバー削除/無効化 | **最高** |
| | 招待リンク | ・招待用URLの発行とコピー<br>・URLの有効期限設定/無効化 | 高 |
| **業務設定** | 稼働カレンダー | ・会社の定休日設定<br>・祝日設定（自動/手動）<br>・標準就業時間（例: 8:00-17:00） | 高 |
| | プロジェクト設定 | ・新規プロジェクト作成時のデフォルト値<br>・アーカイブ済みプロジェクトの管理 | 低 |
| **セキュリティ** | パスワードポリシー | ・最低文字数、文字種制限の設定 | 中 |
| | 監査ログ | ・メンバーの操作ログ閲覧（ログイン、削除等） | 低（将来） |

## 4. Personal Settings (個人設定)
全てのユーザーがアクセス可能。

| カテゴリ | 項目名 | 仕様・機能 | 優先度 |
| :--- | :--- | :--- | :--- |
| **アカウント** | プロフィール | ・表示名 (Display Name)<br>・アイコン画像<br>・自己紹介/職種 | 高 |
| | ログイン情報 | ・メールアドレス変更<br>・パスワード変更<br>・アカウント削除（退会） | **最高** |
| **表示・操作** | テーマ設定 | ・ライトモード / ダークモード / システム準拠<br>・アクセントカラー選択 | 中 |
| | 起動画面 | ・ログイン直後の画面設定（Today / Projects / Dashboard） | 低 |
| **My稼働** | 稼働予定 | ・個人の休暇、早退、残業予定の登録<br>・会社カレンダーとのオーバーライド設定<br>（例：会社は稼働日だが自分は有給） | 高 |
| | 通知設定 | ・メール通知（日報、メンション等）のON/OFF<br>・ブラウザプッシュ通知設定 | 中 |

## 5. 技術的考慮事項
*   **データ構造:** `Tenant` テーブルおよび `UserConfig` テーブル（または `users` テーブルのカラム拡張）で管理。
*   **権限管理:** `CompanySettingsScreen` へのアクセスは、`useAuth()` で取得した `tenant` 情報のroleを確認し、`owner` または `admin` の場合のみ許可する。
*   **API:**
    *   `GET /api/tenant/settings`: 会社設定取得
    *   `PUT /api/tenant/settings`: 会社設定更新
    *   `GET /api/me/settings`: 個人設定取得
    *   `PUT /api/me/settings`: 個人設定更新
