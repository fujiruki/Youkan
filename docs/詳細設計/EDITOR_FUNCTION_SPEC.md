# 建具エディタ (EditorScreen) 機能仕様書

## 1. 概要
建具エディタは、ユーザーが建具の形状、寸法、部材構成を視覚的かつ数値的に編集するための主要画面です。
「Designモード（直感的編集）」と「Proモード（詳細見積・設定）」の2つのビューを提供します。

## 2. データ保存とライフサイクル (Persistance & Lifecycle)

### 2.1 自動保存 (Auto-Save)
ユーザーの操作によるデータ損失を防ぐため、**明示的な保存ボタンを廃止し、常時自動保存**を採用しています。

*   **トリガー**: 建具データ（`door`ステート）の変更を検知
*   **デバウンス処理**: 変更検知から **1.0秒後** にデータベース（IndexedDB）へ書き込み
*   **対象データ**: 寸法、名前、設定など、`door`オブジェクトに含まれる全てのプロパティ

### 2.2 終了時の保存 (Final Save on Exit)
画面遷移時（「戻る」ボタン押下時）には、デバウンスを待たずに**即座に最終状態を保存**します。
この際、後述する「サムネイル画像」の生成と保存も同時に行われます。

## 3. リセットとUndo機能 (Reset & Undo Logic)

編集作業中の「試しにやってみたが、やっぱり元に戻したい」というニーズに応えるため、トグル式のリセット機能を実装しています。

### 3.1 状態管理
エディタは内部的に以下の3つの状態スナップショットを管理します。

1.  **Current State**: 現在編集中のデータ
2.  **Initial Snapshot**: エディタを開いた瞬間のデータ（不変）
3.  **Pre-Reset Snapshot**: リセットを実行する直前のデータ（一時退避用）

### 3.2 動作ロジック

| ユーザー操作 | ボタン表示 | 内部挙動 | 画面状態 |
| :--- | :--- | :--- | :--- |
| **初期状態** | 「リセット (Reset)」 | - | 通常編集 |
| **リセット押下** | -> 「直前 (Undo)」 | 1. 現在の状態を `Pre-Reset Snapshot` に退避<br>2. `Initial Snapshot` を現在の状態に上書き | **初期状態に戻る** |
| **Undo押下** | -> 「リセット (Reset)」 | 1. `Pre-Reset Snapshot` を現在の状態に復元 | **リセット前の編集状態に戻る** |

これにより、ユーザーは「初期状態」と「編集中の状態」をワンクリックで行き来し、比較・検討を行うことができます。

## 4. サムネイル管理 (Thumbnail Management)

一覧画面（Schedule View）の描画パフォーマンスを維持するため、エディタ側で静的な画像を生成・保存します。

### 4.1 生成タイミング
*   **必須**: エディタ終了時（「戻る」ボタン押下時）
*   **任意**: 手動セーブ（もし実装する場合）

### 4.2 生成仕様
*   **ソース**: HTML5 Canvas (`PreviewCanvas`)
*   **形式**: JPEG画像 (Data URL)
*   **内容**: 人型シルエット（スケール感把握用）を含む、現在の建具プレビュー
*   **保存先**: IndexedDB (`Door`テーブルの `thumbnail` カラム)

### 4.3 一覧画面での利用
一覧画面では、重たいCanvas描画コンポーネントの代わりに、この保存されたJPEG画像を表示します。
これにより、建具枚数が増加してもリストのスクロールやレンダリングが遅延しません。

## 5. UI/UX仕様

### 5.1 ヘッダー (Header)
*   **タイトル編集**: 建具名をクリックしてインライン編集が可能。
*   **モード切替**: Design / Pro のタブ切り替え。
*   **リアルタイム見積**: 編集に合わせて単価（Unit Price）を即時再計算して表示。
*   **リセットボタン**: 前述のトグル式ボタン。

### 5.2 プレビュー (Preview Canvas)
*   **人型表示**: デフォルトでON。160cmのシルエットを表示し、サイズ感を直感的に伝達。
*   **操作**:
    *   **ズーム**: マウスホイール（カーソル位置中心）
    *   **パン（移動）**: マウス中ボタン（ホイール）ドラッグ
*   **インタラクション**: 部材（框、組子など）をクリックして選択可能。

### 5.3 ミニエディタ (Mini Editor)
*   **出現位置**: クリックした部材の近くにポップアップ。
*   **ドラッグ移動**: タイトルバーを掴んで自由に配置変更可能。
*   **入力**: 数値入力に特化し、不要な余白を排除したコンパクト設計。
