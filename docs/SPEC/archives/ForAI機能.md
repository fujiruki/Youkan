# ForAI機能

## 概要

・AIへ、業務状況を伝えやすい形式に。
    抱え込んでいるTODOリストの受け渡しがしやすい。
    好みのAIチャットにあなたの現在状況を渡せる！相談できる！

抱えているタスク・プロジェクトの状況（名前や納期、目安期間など）をAIに渡すことができる。これによって、AIはあなたの状況を理解した上で、今後の相談が可能になる。

## 使い方
ForAIボタンを押すとForAIモーダルが開く。
モーダルの構成
    ・現在の立場ラベル
    ・小さいテキストエリア（中身は見切れていて良い）
    ・クリップボードへコピーボタン
    ・ダウンロードボタン

現在のモード・フィルタで表示しているアイテムたちをすべて取得し、
それらのタイトル・プロジェクト構造・アイテムの親子関係・納期・マイ期限・目安時間・目安期間・メモ・担当者・所属会社・所属プロジェクトなどすべての情報をAIに渡しやすい形式で小さいテキストエリアに表示。
「クリップボードへコピー」ボタンを押すと、それをクリップボードに取得できる。
「ダウンロード」ボタンで、そのmdファイルをダウンロードできる。
