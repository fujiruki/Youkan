今直近でやるべきタスクをスマートフォンで読み上げる機能を作りたい。スマホの読み上げ機能を使う前提にしたいかな。お金をかけたくないから。そういう機能を企画してくれない？
「あー今なにをしないといけないんだっけ。つかれたーぼんやりしている。タスクを指示してくれる秘書がいたらいいな」というときに、タスクを読み上げてくれる人がいるとうれしい。


スマホで読み上げ文章が多すぎるとき、読み上げ画面を消せなくなる。スクロールもできない。画面固定の一時停止・再生・１０秒戻るボタンもほしい。


読み上げのとき、１件ずつ字幕のように表示するより、リストで表示して、歌詞表示のようにスクロールしながら表示してほしい。

# 読み上げ順序について
完了済みは読み上げなくていいです。
現在は以下の優先順位で読み上げて（ViewModelのデータ構造に基づいています）：

現在実行中のタスク
今日やる（確定済み）
今日やる（候補・Focus）
Inbox（未分類）
Ready（準備完了）
保留（Pending）
☓完了済み（Log）の履歴

それぞれのグループの中で、{目安期間の初日, マイ期限, 納期} のどれか早い順に並べてほしい。
